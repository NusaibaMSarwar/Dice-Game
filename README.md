A simple dice game built with HTML, CSS, and JavaScript where two players roll dice and the highest number wins!

## How to Play

1. Open `index.html` in your browser
2. Refresh the page to roll the dice
3. The player with the higher number wins!
4. If both dice show the same number, it's a draw

## Features

- Random dice rolls (1-6)
- Automatic winner detection
- Simple and clean interface

- HTML
